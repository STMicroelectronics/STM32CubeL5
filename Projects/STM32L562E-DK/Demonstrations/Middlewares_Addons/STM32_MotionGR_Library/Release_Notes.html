Release Notes for MotionGR Gesture Recognition Library

Copyright 2019 STMicroelectronics

What is MotionGR?

  • The MotionGR provides real-time information about the gesture which the user just performed with the device, i.e. cell phone.
  • The library acquires data from the accelerometer and provides information about the gesture which the user just performed with the device. It is able to distinguish following gestures: pick up, glance, wake up.
